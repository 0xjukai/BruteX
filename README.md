# BruteX

## Automatically brute force all services running on a target

* Open ports
* Usernames
* Passwords

## INSTALL:
```
./install.sh
```

## USAGE:
```
brutex target <port>
```
## DOCKER:
```
docker build -t brutex .
docker run -it brutex target <port>
```
